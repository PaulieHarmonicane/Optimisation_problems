### Для запуска работы необходимо сделать следующее:
1) Для копирования данного проекта себе на комп IDE в терминале введете: </br>
   git clone https://github.com/PaulieHarmonicane/Optimisation_problems.git</br>
   cd Optimisation_problems

2) Теперь необходимо создать окружение проекта. В терминале:</br>
   python3 -m venv .venv</br>
   **Для Unix**: source .venv/bin/activate</br>
   **Для Windows**: venv\Scripts\activate</br>

3) Установите необходимые библиотеки и расширения:</br>
   pip install -r requirements.txt</br>

4) В терминале IDE введите: jupyter notebook</br>

5) ГОТОВО!!! Можено изучать проект!
